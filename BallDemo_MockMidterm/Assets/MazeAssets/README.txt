Project 2 - Character Controllers
(Chantel Gee - chmagee)

 - Keyboard Controls: player uses WASD
 - Navigation and Clipping: cannot through walls of maze
 - Collectables: cubes (aka keys)
	- UI change: "Keys Left : " in top left
 - 3rd Person View: top down view of maze following the player
 - 1st Person View: view from capsule player perspective
	- UI to switch cameras: press "c" to switch views

 - Unique Additiion: if there are 2 keys, the yellow gate stays closed. if
	there is 1 key, the yellow gate opens but the blue gate stays
	closed. if there are no keys, the blue gate opens. gates are also 
	coordinated by color of cube/key
